PhotoImport

Import photos from a camera to daily, nameable folders.

Welcome to the PhotoImport page! PhotoImport is an application that imports pictures form a source (usually a camera or media card) into a destination folder.
Please keep in mind that PhotoImport is currently in early alpha so it can not be expected to work as advertised. Read more about the motivation behind this project on the About page.
Bugs and feature requests should be directed to the issue tracker. A mailing list (currently mostly for development talk) can be found at photoimport-devel@googlegroups.com, in case of interest please register at the Google groups page.
Have a look at the Development Wiki about hacking on PhotoImport.

Dependencies

Python >= 2.6, wxPython and pyexiv2 >=0.2.0.

Install

This project is in early alpha, install instructions will be updated when necessary.

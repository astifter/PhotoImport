PhotoImport

Import photos from a camera to daily, nameable folders.

About

PhotoImport was created by Andreas Neustifter to simplify and speedup the process of importing photos from a camera oder media-card.
Most photomanagement software has its own built in importer, but those are usually not suited for using them without the photomanagement feature. Besides that I wanted to make an importer thats fast and easy to use but highly configurable so that it can be tweaked to all workflows.

get the source code on GitHub : astifter/PhotoImport
